Dijkstra's Algorithm


Dijkstra's algorithm is a widely-used pathfinding algorithm in computer science. It efficiently finds the shortest path between two nodes in a graph by iteratively selecting the node with the lowest cumulative cost from a source node. Although primarily designed for non-negative edge weights, its simplicity and effectiveness make it a foundational tool in network routing, transportation planning, and various other applications requiring optimal path determination. 
Dijkstra's algorithm maintains a priority queue to systematically explore and update the costs of neighboring nodes, ensuring an accurate and optimal path. While its effectiveness is notable, the algorithm may not handle graphs with negative edge weights correctly, and variations like the Bellman-Ford algorithm are preferred for such scenarios.

Bezier Curves

For smooth traversal in the path, the hard edges need to be smoothened out. This is done by calculating the points for a Bezier Curve  between any 2 neighbor nodes, and giving in a step value for how smooth the curve needs to be.
