Place your custom game screenshots in this folder using these filenames:

- raftaar-run.jpg
- balloon-battle.jpg
- snack-street-showdown.jpg

Showcase project background images:

- showcase-raftaar-run.jpg
- showcase-funwave-studio.jpg
- showcase-gameplay-highlights.jpg

You can also use PNG or WEBP if you update the matching `src` values in `index.html`.
If a game card image is missing, the site will automatically show the built-in fallback artwork.
